▶ เนื้อหาครบจบในเกมเดียว

มีด่านย่อยให้ไล่เก็บคะแนนมากกว่าสิบยูนิตต่อระดับ สามารถเลือกหน่วยที่สนใจก่อนได้ และมี Progress บอกว่าคุณเล่นไปถึงด่านไหนแล้ว

ระดับความยากของเนื้อหา JLPT

  • คำศัพท์ JLPT N5 ถึง N1
  • การฟังประโยคมาตรฐาน JLPT ทุกระดับ
  • ทำระบบบททดสอบตามแบบข้อสอบ

รวมคอร์สพิเศษเช่น วัฒนธรรมโอตาคุและการท่องเที่ยว

สาวแกล (Gyaru)

「推し活、一緒に楽しもうね！」
มาสนุกกับการทำกิจกรรมติ่งด้วยกันนะ!

เกมนี้ไม่ได้มีเพียงคำศัพท์ทั่วไป แต่ยังมีคอร์ส 'กิจกรรมโอตาคุ (Oshikatsu)' ที่สอนคำแสลงบน Social Media ไปจนถึงสถานการณ์ไปคอนเสิร์ตที่ญี่ปุ่น

▶ Oshikatsu & Survival Travel

นอกจากนั้นยังมีคอร์สท่องเที่ยว (Travel Survival) สอน 50 คำศัพท์ยามคับขัน ร้านอาหารและการขอความช่วยเหลือเมื่อป่วย

ตัวอย่างหลักสูตรสถานการณ์สมมติ

  1. คอร์สร้านอาหารและเมนูอาหาร
  2. คอร์สการแจ้งอาการป่วย (Medical)
  3. คอร์สเอาตัวรอดจากการหลงทาง

โหมดฝึกการฟังระดับสูงและเฉพาะด้าน

สาวแกล (Gyaru)

「ビジネスの言葉も大事だよ。」
คำศัพท์ธุรกิจก็สำคัญเหมือนกันนะ

เพื่อเน้นย้ำความสามารถในการฟัง ตัวเกมแยกหลักสูตร Listening ออกมาอย่างชัดเจน ครอบคลุมสถานการณ์แบบ Business, ตัวเลขและเวลา, และการโต้ตอบรายวัน

▶ ธุรกิจ, ตัวเลข, และรับมือภัยพิบัติ

ผู้เรียนจะได้รู้วิธีการทำความเข้าใจรูปประโยคที่ซับซ้อน เช่น การรับมือเมื่อเกิดสถานการณ์ภัยพิบัติ (Disaster Survival) ซึ่งคนอาศัยอยู่ญี่ปุ่นต้องรู้

หลักสูตรทางฝั่งการฟังเสียง

  • ฝึกฟังไวยากรณ์สุภาพ (Business language)
  • เรียนรู้การประกาศเตือนภัยในญี่ปุ่น
  • ฝึกแยกแยะคำศัพท์บอกเวลา

พร้อมลุยไปกับเสียงดนตรีประกอบผ่อนคลาย

สาวแกล (Gyaru)

「音楽を聴きながらリラックスしてね。」
ฟังเพลงแล้วผ่อนคลายไปด้วยนะ

เพราะการเรียนในความเงียบอาจทำให้คุณเบื่อ เกมมีระบบ BGM เพลงคลอแนวผ่อนคลายที่จะช่วยเพิ่มสมาธิและทำให้ความทรงจำของคุณผูกกับเพลง

▶ ลดความตึงเครียดผ่านเสียงซาวต์แทร็ค

คุณสามารถกดเริ่มเพลง ปิดเสียง หรือลากปรับระดับวอลลุ่มหน้าแรกได้ตามต้องการเมื่ออยู่ใจกลางเมืองที่มีเสียงดังรบกวน

ตั้งค่า BGM

  • ใช้เพลงสไตล์ Lo-Fi/Relaxing
  • ปิด-เปิด BGM ได้ง่ายดาย
  • สมาธิดีขึ้นเมื่อเรียนพร้อมเสียงเพลงเบาๆ

ฝึกด้วยระบบอ่านออกเสียงอัตโนมัติ (TTS)

สาวแกล (Gyaru)

「私の声、ちゃんと聞こえてる？」
ได้ยินเสียงของฉันชัดไหมคะ?

ตัวเกมเชื่อมต่อกับระบบ Web Speech API เพื่อให้มีเสียงพูด (Text-to-Speech) คลอรอบบทสนทนาภาษาญี่ปุ่น ช่วยให้เข้าใจการออกเสียงอย่างเป็นธรรมชาติ

▶ ฟังสำเนียงจริง ปรับระดับความเร็วได้

นอกจากนี้ ผู้ใช้สามารถเข้าไปที่หน้าต่างตั้งค่า (Settings) บนหน้าแรกเพื่อปรับความเร็วเสียง (Rate) ให้ช้าลง และปรับระดับเสียง (Volume) ให้เหมาะกับการฟังของตัวเอง

หน้าต่างการตั้งค่าเสียง

  1. เสียงภาษาญี่ปุ่นทุกประโยค
  2. ปรับความเร็ว (Rate) และระดับความดังได้
  3. ฝึกทักษะหูจากเจ้าของภาษา(จำลอง)
