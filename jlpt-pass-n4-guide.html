ยกระดับไวยากรณ์สู่ JLPT N4

ถึงจะเนื้อหาเยอะ แต่สาวแกลในเกมจะช่วยให้คุณทบทวนคำช่วยและรูปประโยคต่างๆ แบบเป็นขั้นตอน

Start Game

ทำซ้ำประโยคเดิมให้ขึ้นใจ (Spaced Repetition)

สาวแกล (Gyaru)

「えっと…間違えちゃったね。」
เอ่อ... ตอบผิดซะแล้วสิเนี่ย

การผันรูป て (Te-form) หรือรูปเงื่อนไข มักทำคนปวดหัวใน N4 เกมของเราใช้การทำซ้ำในคำซ้ำๆ เพื่อให้คุณเกิดความเคยชินอัตโนมัติ

▶ จำได้ไม่มีลืม

ตอนตอบผิด สาวแกลจะหงอย ทำให้คุณจำข้อผิดพลาดนั้นได้ฝังใจและแก้ไขในรอบถัดไป

ความพิเศษของควิซเกม

  • จำง่ายผ่านปฏิกิริยาของตัวละคร
  • รูปแบบปรนัยเลือกข้อถูก
  • โฟกัสที่ไวยากรณ์ที่เป็นรากของ N4

รับมือกับ Communication และการตอบรับ

สาวแกล (Gyaru)

「相槌がすごく上手だね！」
ตอบรับเก่งมากๆ เลยนะ!

นอกเหนือจากควิซ เรามีคอร์ส Communication เพื่อรับมือกับการถามสั้นตอบสั้น ซึ่งเป็นไม้ตายของข้อสอบ N4 ในช่วงแรกของพาร์ทฟัง

▶ ไหวพริบใน N4

เกมตั้งโจทย์ให้คุณพิจารณาว่าอีกฝ่ายถามมาแบบนี้ ควรใช้คำตอบไหนให้เป็นธรรมชาติที่สุด

คอร์สการตอบรับรวดเร็ว

  1. เน้นเรื่องปฏิเสธหรือตอบรับข้อเสนอ
  2. ลดอาการงงเวลาได้ยินประโยคใหม่
  3. จบหลักสูตรได้ในไม่กี่นาที

แปลไทยคู่กับฮิรางานะและโรมาจิแบบละเอียด

สาวแกล (Gyaru)

「なるほど、そういうことか！」
อ้อ เข้าใจแล้ว มันเป็นแบบนี้นี่เอง!

ข้อใดที่คุณสงสัย ในเกมคุณทำแค่ชำเลืองตาลงข้างกล่องแชต ก็จะเห็นการแปลทับบรรทัดอย่างละเอียดชัดเจน

▶ ลดภาระการวิเคราะห์

มันช่วยให้จับผิดได้ว่า ไวยากรณ์ 'A ไม่ใช่ B' ในประโยคถูกเปลี่ยนตรงคำใด จึงสะดวกต่อการจดโน้ตแบบเร็วๆ

คู่มือในแต่ละบรรทัด

  • ความแม่นยำสูงในระดับคำศัพท์
  • ช่วยแยกวิเคราะห์รูปอดีตและปัจจุบัน
  • ซับพอร์ตให้คุณเก่งขึ้นอย่างรวดเร็ว

ตั้งสติกับ TTS เสียงอ่านที่ปรับยืดหยุ่น

สาวแกล (Gyaru)

「ゆっくり喋るね。聞いて。」
จะพูดช้าๆ นะ ตั้งใจฟังล่ะ

เสียงพูดที่ขับเคลื่อนด้วย Web Speech API สามารถลดสปีดได้ตลอดเวลาเมื่อคุณรู้สึกว่า N4 มันพูดยาวและซับซ้อนไป

▶ ฟังคำสั้นๆ ที่มักโดนหลอก

เป็นการช่วยพยุงไม่ให้ผู้เล่นเครียดหนักเวลาฟังและสามารถก้าวข้ามกำแพงความกลัวได้ด้วยตัวเอง

วอลลุ่มและสปีด

  • ปรับสปีดลากแถบตรง Settings
  • ถ้าชินแล้วก็ปรับเร็วขึ้นมาระดับปกติ 1.0
  • เสียงสำเนียงช่วยกรองความถูกต้อง

ภาพวิดีโอเกมช่วยเพิ่มมิติการตัดสินใจ

สาวแกล (Gyaru)

「ここは教室だよ、集中して。」
ที่นี่ห้องเรียนนะ ตั้งใจหน่อย

เวลาอ่านข้อสอบกระดาษ คุณต้องจินตนาการ แต่การเล่นเกม คุณมีภาพตัวละครยืนอยู่ในห้องเรียน ทำให้ดึงอินเนอร์ตอบช้อยส์ได้เฉียบแหลม

▶ เสริมสร้างสัญชาตญาณ

การรวมประสาทสัมผัสแบบนี้จะส่งผลดีเยี่ยมในห้องสอบจริง เพราะคุณสามารถนึกภาพสาวแกลขึ้นมาในหัวตอนทำ N4

การอ่านข้อสอบพร้อมอินเนอร์

  1. เห็นว่าใครคุยกับใคร
  2. บรรยากาศสถานที่ช่วยบอกใบ้สรรพนาม
  3. ยกระดับสมองให้จำแบบ 4D
