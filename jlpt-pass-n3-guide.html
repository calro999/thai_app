เข้าสู่สนามสอบขั้นกลางด้วยคลังคำศัพท์หลายสิบบทที่ให้คุณฝึกฝนตนเองในรูปแบบของเกมจีบสาวซิมูเลชัน

เข้าสู่ตัวเกม (Start Game)

ฝึกด้วยระบบอ่านออกเสียงอัตโนมัติ (TTS)

สาวแกล (Gyaru)

「私の声、ちゃんと聞こえてる？」
ได้ยินเสียงของฉันชัดไหมคะ?

ตัวเกมเชื่อมต่อกับระบบ Web Speech API เพื่อให้มีเสียงพูด (Text-to-Speech) คลอรอบบทสนทนาภาษาญี่ปุ่น ช่วยให้เข้าใจการออกเสียงอย่างเป็นธรรมชาติ

▶ ฟังสำเนียงจริง ปรับระดับความเร็วได้

นอกจากนี้ ผู้ใช้สามารถเข้าไปที่หน้าต่างตั้งค่า (Settings) บนหน้าแรกเพื่อปรับความเร็วเสียง (Rate) ให้ช้าลง และปรับระดับเสียง (Volume) ให้เหมาะกับการฟังของตัวเอง

หน้าต่างการตั้งค่าเสียง

  • เสียงภาษาญี่ปุ่นทุกประโยค
  • ปรับความเร็ว (Rate) และระดับความดังได้
  • ฝึกทักษะหูจากเจ้าของภาษา(จำลอง)

รวมคอร์สพิเศษเช่น วัฒนธรรมโอตาคุและการท่องเที่ยว

สาวแกล (Gyaru)

「推し活、一緒に楽しもうね！」
มาสนุกกับการทำกิจกรรมติ่งด้วยกันนะ!

เกมนี้ไม่ได้มีเพียงคำศัพท์ทั่วไป แต่ยังมีคอร์ส 'กิจกรรมโอตาคุ (Oshikatsu)' ที่สอนคำแสลงบน Social Media ไปจนถึงสถานการณ์ไปคอนเสิร์ตที่ญี่ปุ่น

▶ Oshikatsu & Survival Travel

นอกจากนั้นยังมีคอร์สท่องเที่ยว (Travel Survival) สอน 50 คำศัพท์ยามคับขัน ร้านอาหารและการขอความช่วยเหลือเมื่อป่วย

ตัวอย่างหลักสูตรสถานการณ์สมมติ

  1. คอร์สร้านอาหารและเมนูอาหาร
  2. คอร์สการแจ้งอาการป่วย (Medical)
  3. คอร์สเอาตัวรอดจากการหลงทาง

เรียนรูปแบบเกมทางเลือก Visual Novel

สาวแกล (Gyaru)

「一緒に勉強しようよ！」
มาเรียนด้วยกันเถอะ!

เกมนี้ถูกออกแบบมาแบบนิยายภาพ (Visual Novel) ผู้เล่นจะได้พูดคุยและโต้ตอบกับตัวละครสาวแกล ให้ความรู้สึกว่ากำลังเล่นเกมจีบสาวไปพร้อมกับเรียนภาษาญี่ปุ่น

▶ สร้างความสนุก ไม่เหมือนอ่านหนังสือ

คุณไม่ต้องท่องตำราเป็นหน้าๆ เพียงแค่อ่านเนื้อเรื่อง ฟังเสียง และเลือกตอบคำถาม หากทำได้ดีจะได้รับคะแนนสะสมในแต่ละด่าน

การโต้ตอบกับเกม

  • ตัวละครหลักเป็นสาวแกลช่วยสอน
  • มีคำแปลภาษาไทยควบคู่เสมอ
  • กดซ้ำเพื่อฝึกฟังได้ตามต้องการ

มีฟังก์ชันโหมดควิซและโหมดการเรียน

สาวแกล (Gyaru)

「クイズ、全問正解できるかな？」
จะตอบควิซถูกหมดทุกข้อไหมนะ?

ก่อนเริ่มแต่ละหน่วย (Unit) ของบางหลักสูตร ผู้สอนจะให้เลือกโหมดระหว่าง 'โหมดการเรียน (Learning)' ที่เน้นอธิบายเนื้อหา และ 'โหมดควิซ (Quiz)' ที่จะหักคะแนนหากตอบผิด

▶ Learning Mode & Quiz Mode

คุณสามารถกดเข้าไปดูเฉพาะรูปประโยคแบบไม่ต้องกังวลเรื่องเวลาใน Learning Mode หรือท้าทายตนเองใน Quiz Mode ก็ได้ผลลัพธ์ที่ดีทั้งคู่

ความแตกต่างของ 2 โหมด

  • लर्निंगโหมดสำหรับการเริ่มจำ
  • ควิซโหมดวัดความเข้าใจ
  • สามารถกลับไปทบทวนได้ทันที

ระบบปลดล็อกชุดพิเศษ (Outfit Shop)

สาวแกล (Gyaru)

「この服、どうかな？似合う？」
ชุดนี้เป็นยังไงบ้าง? เข้ากับฉันไหม?

คุณสามารถเข้าไปที่เมนู Outfit Shop ในเกมเพื่อเปลี่ยนชุดให้กับสาวแกลได้ โดยมีทั้งชุด Default ฟรี และชุดพรีเมียมอย่าง Yukata, Maid, Miko และ Gothic

▶ รับรางวัลเพื่อเปิดตู้เสื้อผ้า 24 ชั่วโมง

ชุดแต่งกายพรีเมียมสามารถปลดล็อกได้ต่อเนื่อง 24 ชั่วโมง เพียงแค่สนับสนุนเกมผ่านการรับชมโฆษณา ช่วยให้เพลิดเพลินกับการเปลี่ยนชุดระหว่างเรียน

ชุดที่สามารถปลดล็อกได้

  1. ชุดเมด (Maid)
  2. ชุดมิโกะ (Miko)
  3. ชุดกอธิคโลลิต้า (Goth) และ ยูคาตะ (Yukata)
