Start Game

คอร์สโอตาคุ (Oshikatsu) เพื่อสายอิน

สาวแกล (Gyaru)

「推しの話、たくさんしようね！」
มาคุยเรื่องเมนกันเยอะๆ เถอะนะ!

ไม่มีอะไรจะดีไปกว่าการเรียนจากสิ่งที่ชอบ เกมนี้มีคอร์สพิเศษเช่น คำชื่นชมศิลปิน หรือคำน่ารักๆ ที่มักเจอในโลกโซเชียล

▶ เรียนสแลงใน X/Twitter

คุณจะได้รู้ประโยคที่ไม่เคยสอนในหนังสือเรียน และช่วยให้คุยกับเพื่อนชาวญี่ปุ่นได้อย่างเป็นกันเองมากขึ้น

หลักสูตรพิเศษ

  • ประโยคบอกรักเมน
  • สแลงทันสมัยในวงการไอดอล
  • ควิซแนวอนิเมะ

โหมดเครื่องแต่งกาย (Costume) ให้เข้าธีม

สาวแกล (Gyaru)

「新しい服、どう？似合ってる？」
ชุดใหม่เป็นยังไงบ้าง เข้าไหม?

เพื่อตอบสนองคนรักอนิเมะ ระบบเสื้อผ้า (Outfit) ช่วยให้คุณเปลี่ยนสาวแกลที่หน้าเมนูหลักได้ตลอดเวลา

▶ ชุดเมดถึงยูคาตะ

คุณอาจให้เธอใส่ชุดยูคาตะตอนกลางคืน และเปลี่ยนเป็นชุดเมดในตอนเช้า สร้างบรรยากาศฮาเร็มนิดๆ ระหว่างเรียน

การปลดล็อกรายวัน

  1. ปลดล็อกฟรีแค่เชยชมสนับสนุนแอป
  2. โหมดจำค่าชุด ไม่ต้องเปลี่ยนใหม่บ่อย
  3. มีดีไซน์ภาพแบบฟูลสเกลหลายแบบ

ประโยคซับไววัลสำหรับคนบินบ่อย (Travel)

สาวแกล (Gyaru)

「日本への旅行、準備はバッチリ？」
เตรียมตัวไปเที่ยวญี่ปุ่นพร้อมหรือยัง?

สายเจแปนเลิฟเวอร์ที่ไปญี่ปุ่นบ่อยๆ เรามีหมวด Travel Survival 50 คำศัพท์ ที่สรุปคำพูดของพนักงานเซเว่นไปจนถึงพนักงานสถานีรถไฟ

▶ เอาตัวรอดจากการหลงทาง

มีคอร์สเจาะลึกการใช้บริการทางการแพทย์ (Medical) อย่างการซื้อยายามป่วย ซึ่งเป็นข้ออมตะที่ทุกคนต้องเจอ

คอร์สฉุกเฉิน

  • บทสนทนาสถานีรถไฟ
  • บทพูดตามร้านขายยา
  • บทสั่งอาหารร้านบุฟเฟต์

ภาพและฟีลลิ่งเกมซิมูเลชัน (Visual Novel)

สาวแกล (Gyaru)

「私と一緒に、楽しんでね！」
มาสนุกไปกับฉันนะ!

UI ได้รับการออกแบบแนว Glassmorphism ที่ตัดกับพื้นหลังห้องเรียนหรือสถานีรถไฟ สื่อถึงเกมล้ำยุคที่ผสมผสานไลท์โนเวล

▶ ออกแบบเพื่อคนรัก 2D

การดำเนินเรื่องผ่านกล่องข้อความบรรทัดต่อบรรทัด ทำให้สมองลดภาระความตึงเครียด ได้ดื่มด่ำไปกับฉากและคาร์แรกเตอร์

อินเตอร์เฟซสไตล์ญี่ปุ่น

  • ธีมเกมถูกวางสัดส่วนลงตัวบนมือถือ
  • ดีไซน์ปุ่มกดตอบอย่างโดดเด่น
  • พื้นหลังมีความเกี่ยวข้องกับสิ่งที่เรียน

คลิกลิงก์ เล่นผ่านเบราว์เซอร์ได้ 100%

สาวแกล (Gyaru)

「いつでも会いに来てね！」
แวะมาหาฉันได้ตลอดเวลาเลยนะ!

แค่จับมือถือตอนพักเที่ยง คุณก็วาร์ปเข้ามาดูอัปเดตของเกมได้ ไม่ต้องกังวลเรื่องการกินความจำมือถือ เพราะใช้ระบบ Web App HTML5 เต็มรูปแบบ

▶ ยกระดับ Web-App

คะแนนและสถิติจะถูกจัดเก็บบนสเตอเรจเครื่องอย่างปลอดภัย หายห่วงเรื่องความสะดุดในการใช้งาน

ไม่ต้องพกความจุหนัก

  1. รองรับ iOS และ Android ทุกรุ่น
  2. ประหยัดแบตเตอรี่เครื่อง
  3. กลับมาเล่นค้างได้อัตโนมัติ
