ถ้าคุณชอบเคลียร์เกมให้ได้ Perfect Score คุณมีความสามารถพอที่จะผ่านเกณฑ์ข้อสอบ JLPT ให้ชัวร์แบบไม่กดดัน

เข้าสู่ตัวเกม (Start Game)

โหมดฝึกการฟังระดับสูงและเฉพาะด้าน

สาวแกล (Gyaru)

「ビジネスの言葉も大事だよ。」
คำศัพท์ธุรกิจก็สำคัญเหมือนกันนะ

เพื่อเน้นย้ำความสามารถในการฟัง ตัวเกมแยกหลักสูตร Listening ออกมาอย่างชัดเจน ครอบคลุมสถานการณ์แบบ Business, ตัวเลขและเวลา, และการโต้ตอบรายวัน

▶ ธุรกิจ, ตัวเลข, และรับมือภัยพิบัติ

ผู้เรียนจะได้รู้วิธีการทำความเข้าใจรูปประโยคที่ซับซ้อน เช่น การรับมือเมื่อเกิดสถานการณ์ภัยพิบัติ (Disaster Survival) ซึ่งคนอาศัยอยู่ญี่ปุ่นต้องรู้

หลักสูตรทางฝั่งการฟังเสียง

  • ฝึกฟังไวยากรณ์สุภาพ (Business language)
  • เรียนรู้การประกาศเตือนภัยในญี่ปุ่น
  • ฝึกแยกแยะคำศัพท์บอกเวลา

หลักสูตรรองรับตั้งแต่ระดับพื้นฐานจนถึง N1

สาวแกล (Gyaru)

「試験の準備は完璧？」
เตรียมพร้อมสำหรับการสอบหรือยัง?

คำศัพท์และหลักไวยากรณ์ถูกแบ่งตามมาตรฐานการสอบ JLPT ตั้งแต่ระดับผู้เริ่มต้นระดับ N5 ไปจนถึงระดับสูงสุดอย่าง N1 ซึ่งคอร์ส N5 ถึง N3 นั้นเข้าเล่นได้ฟรี

▶ เนื้อหาครบจบในเกมเดียว

มีด่านย่อยให้ไล่เก็บคะแนนมากกว่าสิบยูนิตต่อระดับ สามารถเลือกหน่วยที่สนใจก่อนได้ และมี Progress บอกว่าคุณเล่นไปถึงด่านไหนแล้ว

ระดับความยากของเนื้อหา JLPT

  1. คำศัพท์ JLPT N5 ถึง N1
  2. การฟังประโยคมาตรฐาน JLPT ทุกระดับ
  3. ทำระบบบททดสอบตามแบบข้อสอบ

จำอักษรฮิรางานะและคาตาคานะด้วย Piano Mode

สาวแกล (Gyaru)

「あいうえお、言えるかな？」
อะ อิ อุ เอะ โอะ พูดได้ไหมนะ?

สำหรับคนยังจำตัวอักษรพื้นฐานไม่ได้ ไม่ใช่ปัญหา! ภายในหลักสูตรฮิรางานะและคาตาคานะ มี Piano Mode ที่ให้คุณคลิกแป้นเหมือนเปียโนเพื่อฟังเสียงอ่าน

▶ สร้างสัมพันธ์เสมือนเสียงดนตรี

ช่วยให้สร้างความจำด้านเสียง (Audio Memory) โดยไม่ต้องท่องจากกระดาษ และสามารถนำไปเชื่อมโยงกับการอ่านในด่านอื่นๆ ได้ง่ายขึ้น

หลักการทำงานของโหมดเปียโน

  • กดได้เหมือนเปียโน
  • เสียงอ่านชัดเจนทุกตัวสะกด
  • ช่วยให้มือใหม่จำได้ง่าย

มีฟังก์ชันโหมดควิซและโหมดการเรียน

สาวแกล (Gyaru)

「クイズ、全問正解できるかな？」
จะตอบควิซถูกหมดทุกข้อไหมนะ?

ก่อนเริ่มแต่ละหน่วย (Unit) ของบางหลักสูตร ผู้สอนจะให้เลือกโหมดระหว่าง 'โหมดการเรียน (Learning)' ที่เน้นอธิบายเนื้อหา และ 'โหมดควิซ (Quiz)' ที่จะหักคะแนนหากตอบผิด

▶ Learning Mode & Quiz Mode

คุณสามารถกดเข้าไปดูเฉพาะรูปประโยคแบบไม่ต้องกังวลเรื่องเวลาใน Learning Mode หรือท้าทายตนเองใน Quiz Mode ก็ได้ผลลัพธ์ที่ดีทั้งคู่

ความแตกต่างของ 2 โหมด

  • लर्निंगโหมดสำหรับการเริ่มจำ
  • ควิซโหมดวัดความเข้าใจ
  • สามารถกลับไปทบทวนได้ทันที

พร้อมลุยไปกับเสียงดนตรีประกอบผ่อนคลาย

สาวแกล (Gyaru)

「音楽を聴きながらリラックスしてね。」
ฟังเพลงแล้วผ่อนคลายไปด้วยนะ

เพราะการเรียนในความเงียบอาจทำให้คุณเบื่อ เกมมีระบบ BGM เพลงคลอแนวผ่อนคลายที่จะช่วยเพิ่มสมาธิและทำให้ความทรงจำของคุณผูกกับเพลง

▶ ลดความตึงเครียดผ่านเสียงซาวต์แทร็ค

คุณสามารถกดเริ่มเพลง ปิดเสียง หรือลากปรับระดับวอลลุ่มหน้าแรกได้ตามต้องการเมื่ออยู่ใจกลางเมืองที่มีเสียงดังรบกวน

ตั้งค่า BGM

  1. ใช้เพลงสไตล์ Lo-Fi/Relaxing
  2. ปิด-เปิด BGM ได้ง่ายดาย
  3. สมาธิดีขึ้นเมื่อเรียนพร้อมเสียงเพลงเบาๆ
