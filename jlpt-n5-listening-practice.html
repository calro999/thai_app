บทสนทนาประจำวันในข้อสอบ N5 ฟังให้ติดหู

สาวแกล (Gyaru)

「毎日の挨拶が基本だよ！」
การทักทายทุกวันคือพื้นฐานเลยนะ!

การฟัง N5 ส่วนมากจะเป็นเรื่องของคนนัดหมาย คุยครอบครัว หรือซื้อของ เราจับประเด็นนี้มาทำเป็นยูนิตในเกมโดยเฉพาะ

▶ Daily Life Audio

เพื่อให้คุณทดสอบทักษะการดักจับข้อมูลอย่าง จำนวน, วันที่, ตำแหน่ง ได้อย่างไหลลื่น

โหมดฝึกฟัง N5

  • การช้อปปิ้งและสอบถามราคาสินค้า
  • ถามถึงฤดูกาลตลอดปีเป็นภาษาญี่ปุ่น
  • จัดเป็นชุดยูนิต 10 ข้อกำลังดี

อ่านไม่ทัน หรือฟังไม่ออก? TTS ช่วยได้

สาวแกล (Gyaru)

「ゆっくり喋ることもできるよ！」
ฉันพูดช้าลงให้ได้ด้วยนะ!

ฟีเจอร์เด่นที่สุดของเกมนี้ คือฟังก์ชันปรับลดความเร็วของ Web Speech API ทำให้เด็กใหม่ที่เพิ่งเริ่มจำฮิรางานะ ไม่ต้องช็อกเสียงพูดคนญี่ปุ่น

▶ ลดความเร็วลงตามสั่ง

ตั้งค่าให้ช้าลงเพื่อเก็บรายละเอียดทีละคำ และเมื่อหูเปิด ก็ปรับกลับขึ้นมาเป็นปกติ

การใช้ Slider

  1. ลดความเร็ว Rate ได้ต่ำสุดขีด
  2. เสียงอ่านทุกลมหายใจเป็นธรรมชาติ
  3. เป็นอาวุธสำหรับคนที่หูมืดบอด

ดูสคริปต์ภาษาไทย คลายความกังวล

สาวแกล (Gyaru)

「タイ語のスクリプトを確認してね。」
เช็คสคริปต์ภาษาไทยดูสิ

เวลาพลาดไม่รู้ว่าตัวละครบ่นอะไร คุณสามารถเช็คดูที่พื้นที่ส่วนล่างของกล่องบทบรรยายซึ่งพ่วงคำแปลภาษาไทยแสนวิจิตรไว้ให้

▶ แปลชัด ไม่ต้องอ้อม

ทำให้คนฝึกฟัง N5 ที่มีคำศัพท์น้อย ไม่ต้องเปิดแอปอื่นไปมาให้เสียเวลา

กล่องข้อความอเนกประสงค์

  • แปลไทยละเอียดทับไปกับอักษรญี่ปุ่น
  • ตัวละครพูดเมื่อไหร่ แผลเมื่อนั้น
  • สะดวกสุดๆ ต่อการจดโน้ต

โต้ตอบการสนทนา Baby Step 1-2-3

สาวแกล (Gyaru)

「ちょっとずつでいいからね！」
ทีละนิดก็ไม่เป็นไรหรอกนะ!

หาก N5 ยังยากเกินไป เกมเรามีหลักสูตร Baby Step ที่สอนเฉพาะคำง่ายมาก ๆ เช่น 'ยินดีด้วย' หรือ 'โชคดีนะ'

▶ ระดับผู้เริ่มต้นอย่างแท้จริง

ทำให้เกิดสมาธิและการเรียนรู้แบบอิมาร์ชัน (Immersion) ที่จะปูสู่ N5 ได้ไบรท์ที่สุด

คอร์สเริ่มก้าวเดิน

  • คำอุทาน และคำชวนคุยง่าย
  • ช่วยเสริมพลังบวกก่อนลุยยากขึ้น
  • ระบบเรียนค่อยเป็นค่อยไป ไม่กดดัน

ปูทางจิตใจไม่ให้ท้อด้วย Visual Novel

สาวแกล (Gyaru)

「私と一緒に、楽しんでね！」
มาสนุกไปกับฉันนะ!

การฟังซ้ำๆ มันน่าเบื่อ แต่ท่ามกลางห้องเรียน 2D และสาวแกลที่ขยับเขยื้อนแสดงอารมณ์ จะช่วยเติมความสดชื่น

▶ ระบบผูกพันธ์ผู้เล่น

เปลี่ยนความรู้สึกว่า 'ฉันกำลังลุยกับข้อสอบ' กลายเป็น 'ฉันกำลังคุยกับตัวละครน่ารัก'

ความรู้สึกในการเล่นเกม

  1. บรรยากาศเกมสุดผ่อนคลาย
  2. ลดความกดดันจากการซ้อมสอบ N5
  3. สามารถแต่งตัวให้สาวแกลเพิ่มเติมได้
