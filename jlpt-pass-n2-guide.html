ถึงจะเป็นระดับสูง แต่การเอาชนะด้วยการเล่นซ้ำๆ อย่างระบบของคะแนนเกมก็ช่วยให้คุณจำได้ไวขึ้น

Start Game

ล้มบอส N2 ด้วยการผันกริยาขั้นสูง

สาวแกล (Gyaru)

「N2の文法、しっかり区別しよう！」
มาแยกแยะไวยากรณ์ N2 ให้ชัดเจนกันเถอะ!

ข้อสอบ N2 ขึ้นชื่อเรื่องความกำกวมของไวยากรณ์ที่คล้ายกัน เราจึงบรรจุโจทย์ที่ต้องแยกแยะ わけにはいかない กับ わけではない

▶ ไวยากรณ์สับสน

เมื่อคุณตอบถูก คุณจะคุ้นชินรูปแบบโครงสร้าง พอสอบจริงคุณจะตัดช้อยส์ได้อย่างแม่นยำ ไม่ลังเล

ยูนิต N2

  • เคลียร์ด่าน N2 ทุกซอกทุกมุม
  • ทำโจทย์จับผิดไวยากรณ์ที่คล้ายกัน
  • มีคำอธิบายภาษาไทยพร้อมใช้

แยกประเภทคำศัพท์ระดับสูงที่เจอบ่อย

สาวแกล (Gyaru)

「漢字の組み合わせ、覚えてね。」
จำการประสมคันจิเอาไว้ด้วยนะ

คำศัพท์ N2 ไม่ใช่แค่คำโดดๆ แต่มีคันจิประสมที่หน้าตาคล้ายกัน เกมนี้จะใช้ Visual Novel ในการสุ่มควิซให้คุณแยกคันจิ

▶ หมวดคำประสม

เมื่อเล่นบ่อยๆ สมองคุณจะทำงานแบบ Auto-Pilot ทำให้ลดเวลาในพาร์ทอ่านได้อย่างมหาศาล

การจำคำศัพท์ N2

  1. จำศัพท์ผ่านบริบทเกม
  2. อ่านคันจิได้อย่างเป็นธรรมชาติ
  3. สะสมคำศัพท์ไว้ใช้ตอบโจทย์อ่าน

ไม่ต้องซื้อหนังสือแพงๆ ใช้เว็บเพลย์

สาวแกล (Gyaru)

「スマホでもPCでも遊べるよ！」
เล่นได้ทั้งบนสมาร์ทโฟนและพีซีเลยนะ!

ตำราสอบ N2 มักมีราคาสูง แต่โปรเจกต์เกมนี้เปิดให้คุณเล่นและศึกษาเนื้อหาได้ฟรีทุกพาร์ทหลัก

▶ ประหยัดทุนทรัพย์

สามารถเปิดผ่านบราวเซอร์ทั้ง มือถือ และ แท็บเล็ต ช่วยประหยัดค่าใช้จ่ายแถมได้ผลลัพธ์ที่ดีไม่แพ้กัน

แพลตฟอร์ม

  • เล่นฟรีไม่มีค่าใช้จ่ายซ่อนเร้น
  • อัปเดตระบบเกมอย่างสม่ำเสมอ
  • รองรับแพลตฟอร์ม Chrome และ Safari

จำลองสอบ Quiz กะเทาะเปลือก N2

สาวแกล (Gyaru)

「満点取れるまで頑張ろう！」
พยายามจนกว่าจะได้คะแนนเต็มนะ!

หากคุณมั่นใจว่าจำเนื้อหาทั้งหมดได้แล้ว ลองเข้าสู่โหมด Quiz ที่จะหักคะแนนทันทีเมื่อคุณพลาด

▶ หักแต้มคือประสบการณ์

ความกดดันจากโหมดนี้จะจำลองความตื่นเต้นในห้องสอบ ทำให้คุณจัดการอารมณ์และสมาธิได้ดีขึ้น

โหมดท้าทาย

  • ปั่นคะแนนเต็มเพื่อรับถ้วยรางวัล
  • ผิด 1 ครั้งจำตลอดไป
  • แข่งขันกับสถิติของตัวเอง

เสียงพากย์ TTS เคลียร์ปัญหาฟังสำเนียง

สาวแกล (Gyaru)

「聞き取れるかな？」
ฟังออกไหมนะ?

พาร์ทฟัง N2 จะมีความเร็วเฉลี่ยเท่าคนญี่ปุ่นทั่วไปฟังสบายๆ ระบบเกมรองรับการพูดแบบ Natural Speed

▶ TTS สมจริง

หากคุณฟัง 1x ได้คล่อง แสดงว่าหูคุณทะลวงกำแพง N2 ได้แล้ว แต่ถ้าไม่ไหว ก็ปรับลดมาที่ 0.8x ได้

การฟังของ N2

  1. ปรับสปีดได้ตามใจ
  2. เสียงฟังดูสุภาพเป็นธรรมชาติ
  3. เปิดฟังพร้อมดูข้อความไทยได้
