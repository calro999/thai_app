รสชาติและคำคุณศัพท์
Tastes & Textures
あじ (รสชาติ)
💬 คำพื้นฐานที่ใช้พูดถึง 'รสชาติ' ของอาหารโดยรวม
📌 Aji (あじ) = รสชาติ
あっさり (รสสัมผัสเบาๆ / ทานง่าย)
💬 ใช้กับอาหารที่รสชาติไม่จัดจ้าน เกินไป ทานแล้วรู้สึกสดชื่น ไม่เลี่ยน
📌 Assari (あっさり) = รสอ่อนๆ / สดชื่น / ไม่เลี่ยน
こってり (รสเข้มข้น / มัน / เลี่ยน)
💬 ใช้กับอาหารที่มีความมันและรสเข้มข้นสูง เช่น ราเมงซุปกระดูกหมูที่มันๆ
📌 Kotteri (こってり) = รสเข้มข้นและมัน / เลี่ยน
しゅん (วัตถุดิบตามฤดูกาล)
💬 คนญี่ปุ่นให้ความสำคัญกับสิ่งนี้มาก! คือช่วงเวลาที่วัตถุดิบชนิดนั้นอร่อยที่สุดในรอบปี
📌 Shun (しゅん) = ฤดูกาลของวัตถุดิบ (ช่วงที่อร่อยที่สุด)
おいしい (อร่อย)
💬 คำชมที่เชฟญี่ปุ่นอยากฟังที่สุด! อย่าลืมพยักหน้าแล้วพูดคำนี้ตอนทานนะ
📌 Oishii (おいしい) = อร่อย
あつい (ร้อน)
💬 ใช้เตือนตัวเองหรือเพื่อนเวลาทานซุปหรือของทอดที่เพิ่งเสิร์ฟใหม่ๆ
📌 Atsui (あつい) = ร้อน
つめたい (เย็น)
💬 ใช้กับน้ำดื่มเย็นๆ หรือเมนูบะหมี่เย็น
📌 Tsumetai (つめたい) = เย็น
やわらかい (นุ่ม)
💬 ใช้ชมเนื้อสัตว์หรือขนมที่เคี้ยวง่ายและเนียนนุ่ม
📌 Yawarakai (やわらかい) = นุ่ม
かたい (แข็ง)
💬 ใช้บอกความแข็งของเนื้อ หรือใช้สั่งระดับความแข็งของเส้นราเมง
📌 Katai (かたい) = แข็ง
にがて (ไม่ชอบ / ไม่ถนัด)
💬 คำสุภาพที่ใช้บอกว่าเราทานบางอย่างไม่ได้ หรือไม่ชอบรสชาตินั้นๆ
📌 Nigate (にがて) = ไม่ถนัด / ไม่ชอบ (ใช้ปฏิเสธเรื่องอาหารอย่างสุภาพ)

乾杯（かんぱい）

ก่อนจะเริ่มดื่มแก้วแรก คนญี่ปุ่นจะรอให้ทุกคนได้เครื่องดื่มครบก่อน แล้วชนแก้วพร้อมกันว่า 'คัมปาย!' (Kanpai)
