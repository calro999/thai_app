ระดับชั้นกลางต้องระวังเรื่องคำสุภาพและการตีความ ผู้สอนของเราจัดหลักสูตรจำลองเอาไว้เพื่อสิ่งนี้

Start Game

พาร์ทการฟัง N3 คือกำแพงความสุภาพ (Keigo)

สาวแกล (Gyaru)

「ビジネスの言葉も大事だよ。」
คำศัพท์ธุรกิจก็สำคัญเหมือนกันนะ

ข้อสอบ N3 จะรวมเรื่องการฟังบทสนทนาแบบซับซ้อนขึ้น ในบริษัทหรือโรงเรียน คอร์ส Business เราช่วยจำลองได้

▶ เสริมฐานภาษาสุภาพ

คุณจะชินกับการใช้ Kensongo (ถ่อมตัว) และ Sonkeigo (ยกย่อง) ที่เป็นทริคหลักของ N3

หมวดเรียนธุรกิจ Business

  • แปลไทยคำสุภาพได้ชัดเจน
  • จัดระเบียบหูให้ชินกับประโยคยาวเหยียด
  • ยูนิตสั้นๆ ทำทุกวันจนเกิดความจำติดลบ

TTS ไม่รอช้า ให้คุณรันสปีดสม่ำเสมอ

สาวแกล (Gyaru)

「発音、よく聞いてね。」
ฟังการออกเสียงให้ดีนะ

ระดับนี้ ไม่มีเสียงที่พูดช้าลง เกมของเราก็อิงเสียง Web API ให้รันด้วยความเร็ว 1x เป็นหลัก (แต่คุณยังปรับได้นะ)

▶ ลบความช้าจาก N4

ช่วยฝึกการเพ่งเล็งสมาธิในการจับใจความตรงกลางประโยคที่เป็นแก่นสารสำคัญที่สุดของ N3

สปีดเจ้าของเสียง

  1. ให้เสียงเหมือนสถานการณ์คุยจริง
  2. ไม่มีการทวนคำถามซ้ำไปซ้ำมา
  3. วัดความเร็วและไหวพริบ

กล่องเนื้อเรื่องแบบนิยาย ช่วยบอกใบ้บริบท

สาวแกล (Gyaru)

「ここは教室だよ、集中して。」
ที่นี่ห้องเรียนนะ ตั้งใจหน่อย

ปัญหาหลักเวลาทำข้อสอบเทปคาสเซ็ทคือไม่มีภาพ ในเกมตัวละครและหน้าฉากจะช่วยใบ้ได้ว่านี่คือสถานการณ์ไหน

▶ Visual Novel ใช้งานจริง

มันคือการฝึกให้คุณเดาคำศัพท์ N3 ล่วงหน้าจากสิ่งรอบตัว ซึ่งเป็นทฤษฎีการเอาไปใช้เดาในห้องสอบกระดาษแบบยอดเยี่ยม

ความเข้ากันของรูปและเสียง

  • พื้นหลังและบรรยากาศตัวละครดึงฟีลลิ่ง
  • หลุดโฟกัสยากขึ้น เพราะหน้าจอน่ามอง
  • ช้อยส์ทางเลือกแสดงตามกลไกเกม

ให้การแปลไทยบอกส่วนขยายคำนาม (Relative)

สาวแกล (Gyaru)

「なるほど、そういうことか！」
อ้อ เข้าใจแล้ว มันเป็นแบบนี้นี่เอง!

N3 พาร์ทฟังจะเอะอะก็ขยายคำนามยาวเหยียดไปหมด! กล่องแปลภาษาไทยของเราจะตัดโครงสร้างนี้ให้คุณมองลึกซึ้ง

▶ เข้าใจเร็วขึ้นหลายเท่า

เมื่อคุณเห็นภาษาไทยประกอบกับการฟังเสียงพร้อมๆ กัน สมองคุณก็จะคลิกว่าที่คนญี่ปุ่นพูดรัวๆ มันหมายถึงจุดไหน

ความซับซ้อนของส่วนขยาย N3

  • แยกประโยคหน้าประโยคหลังเห็นผลทันตา
  • ฮิรางานะระบุควบคู่กับคันจิออโต้
  • ช่วยร่นเวลาเรียนลงเป็นหลักชั่วโมง

ผลลัพธ์การดวลและสถิติควิซที่เด็ดขาด

สาวแกล (Gyaru)

「基礎からやり直せるよ！」
เริ่มใหม่ตั้งแต่พื้นฐานได้เลยนะ!

เพราะ N3 เป็นเส้นแบ่งความท้อแท้ ระบบจะขึ้นคะแนน 10/10 เมื่อคุณทำยูนิตพาร์ทฟังสำเร็จแบบไร้รอยขีดข่วน

▶ ให้รางวัลกับตัวเองได้ถ้วย Perfect

ความภูมิใจและคะแนนนี้บันทึกลงในไดรฟ์เครื่องคอม/มือถือของคุณ และเป็นเครื่องการันตีให้ตัวเองลงสอบอย่างมั่นใจ

ความรู้สึกแห่งชัยชนะ

  1. กดรีแมตช์หาจุดผิดได้เสมอ
  2. ไม่ต้องกังวลเซิร์ฟเวอร์หลุด เล่นแบบมีอิสระ
  3. โหมด Quiz คือตัวพิสูจน์จุดอ่อนของการฟัง
