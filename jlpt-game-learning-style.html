สาวแกล (Gyaru)

「新しい服、どう？似合ってる？」
ชุดใหม่เป็นยังไงบ้าง เข้าไหม?

การอ่านแต่หน้าข้อศอกสี่เหลี่ยมไม่จูงใจคนเล่น แต่การมีระบบปลดล็อกขุด Miko / Maid จากร้านค้าให้สาวแกลใส่เพื่อเป็นอาจารย์

▶ กระตุ้นลีดมิน

มันคือแรงผลักดันชั้นดีที่ทำให้ผู้ใหญ่และวัยรุ่นอยากเข้ามากดเรียนทุกวันแบบอัตโนมัติ 24 ชั่วโมงต่อการสนับสนุน

ร้านค้าเปลี่ยนชุด Outfit

  • ได้ชุดฟรีด้วยการรับชมโฆษณาสนับสนุนผู้พัฒนา
  • จดจำการใส่ชุดข้ามอุปกรณ์ในเบราว์เซอร์นั้น
  • เปลี่ยนบรรยากาศเป็นฮาเร็มคอมเมดี้สนุกๆ

การดึงสายตาด้วยกราฟิก Visual Novel

สาวแกล (Gyaru)

「えっと…間違えちゃったね。」
เอ่อ... ตอบผิดซะแล้วสิเนี่ย

สมองชอบสิ่งที่เคลื่อนไหว เรามีตัวละครสาวแกลที่เปลี่ยนหน้าตามเนื้อเรื่อง ดีใจ ผิดหวัง ช่วยเป็นสัญลักษณ์เชื่อมโยงการตัดสินใจในควิซ

▶ สร้างความผูกพัน 2D

ถ้าคุณกดโจทย์ผิด คุณจะไม่อยากทำผิดอีกเพราะตัวละครหน้าเศร้า มันคือพลังของ Visual Education 100%

อินเตอร์แอคทีฟดีเอ็นเอ

  1. สกินตัวละครออกแบบ 8 แอกชั่นหลัก
  2. สร้าง Emotional Connection ได้ลึกซึ้ง
  3. เหมือนดูอนิเมะไปพร้อมกับการติว

ไม่ต้องติดตั้ง โหลดปุ๊บเพลย์ปั๊บ (Web-App)

สาวแกล (Gyaru)

「スマホでもPCでも遊べるよ！」
เล่นได้ทั้งบนสมาร์ทโฟนและพีซีเลยนะ!

เราทราบดีว่าคนไม่อยากโหลดแอปใหม่ให้รกเครื่อง เกมของเราทำงานบนเบราว์เซอร์ลื่นไหล เพียงแค่บุ๊กมาร์กลิงก์ไว้

▶ ไร้ข้อจำกัดระบบ

สามารถเปิดค้างไว้เพื่อทำโจทย์ตอบศัพท์บนรถไฟ สลับเป็นจอแนวนอนตอนนั่งคอมที่บ้าน โดยข้อมูลคะแนนยังคงอยู่

เทคโนโลยี HTML5 Canvas

  • ลดสเปคเครื่องมือถือแบบไม่ต้องห่วง
  • เป็นมิตรกับผู้ใช้งาน Safari และ Chrome ทุกระบบรุ่น
  • รักษาความเป็นส่วนตัว ไม่ต้องล็อกอินบัญชีไอดี

คะแนนและสถิติส่วนบุคคล (Local Storage)

สาวแกล (Gyaru)

「満点取れるまで頑張ろう！」
พยายามจนกว่าจะได้คะแนนเต็มนะ!

เมื่อจบแต่ละบท จะมีหน้าต่างสรุปผลคะแนน เช่น 8 จาก 10 ซึ่งบันทึกลงในเครื่อง และมีถ้วยรางวัลรออยู่หากทำได้เต็ม 10

▶ หน้าต่างโชว์ 10/10 Perfect

การไม่มีลีดเดอร์บอร์ดโชว์ชาวบ้าน ทำให้ลดความกดดันทางสังคม มีเพียงคุณและเป้าหมายส่วนตัวของคุณ เพื่อการแข่งขันที่ดีต่อสุขภาพ

Score Achievement

  • เก็บบันทึกแต้มสูงสุดเสมอ
  • วนด่านเพื่อไขว่คว้าถ้วย Perfect
  • สะท้อนสกิลตัวเองก่อนลงสนามสอบ

การฟังเสียง TTS ควบคู่ตัวอักษรไทย

สาวแกล (Gyaru)

「ゆっくり喋るね。聞いて。」
จะพูดช้าๆ นะ ตั้งใจฟังล่ะ

หลักจิตวิทยายืนยันว่า ยิ่งรับรู้ทางตาและหูพร้อมกันจะจำแม่นขึ้น เราใช้ Web Speech API ให้ระบบออกเสียงไดอะล็อกทันทีที่พิมพ์ขึ้นโชว์

▶ Double Coding สมอง

ถ้าเร็วไปก็สโลว์ดาวน์ด้วยแผงวงจรในเมนูหลัก รวมกับคำแปลไทยที่แม่นยำ ทุกส่วนช่วยดันขีดความจำของคนธรรมดาให้ทะลุเพดาน

ฟีเจอร์เพิ่มความจำ

  1. ดึงเอาศักยภาพการจำเสียงมาใช้
  2. ช่วยพังกำแพงอ้างการไม่มีพรสวรรค์ทิ้งไป
  3. เหมาะกับผู้เริ่มต้นตั้งแต่ระดับฮิรางานะ 1-2
