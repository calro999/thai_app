JLPT Gyaru เข้าเล่นเกม

จำลองสถานการณ์การฟัง JLPT N4

พบกับบทสนทนารายวันที่จะโผล่มาหลอกล่อในห้องสอบ เราแยกแยะประเด็นให้เป็นข้อๆ ในโหมด Learning แล้ว

Start Game

เจาะลึก 30 สถานการณ์การฟัง N4 ออริจินัล

สาวแกล (Gyaru)

「N4のリスニング、始まるよ！」
พาร์ทการฟัง N4 จะเริ่มแล้วนะ!

ในเนื้อหาเกมมีหมวด Listening JLPT N4 ที่ถูกจัดแจงไว้ถึง 30 ยูนิตย่อย เพื่อจำลองสถานการณ์การฟังที่ซับซ้อนขึ้นอย่างค่อยเป็นค่อยไป

▶ โหมดฝึกฟังโดยเฉพาะ

ตั้งแต่การบอกให้อีกฝ่ายทำอะไร หรือการขอคำแนะนำ ซึ่งเป็นจุดที่ N4 จะนำมาทดสอบบ่อยที่สุด

หลักสูตร Listening N4

  • กว่า 30 บททดสอบสถานการณ์
  • โต้ตอบการฟังผ่าน Visual Novel
  • ครอบคลุมพาร์ทที่ออกสอบร้อยเปอร์เซ็นต์

ระบบแปลงข้อความเป็นเสียง (Web Speech API)

สาวแกล (Gyaru)

「私の声、ちゃんと届いてる？」
เสียงของฉันส่งไปถึงอย่างชัดเจนไหม?

ระบบเบื้องหลังคือ Web Speech API ที่เรียกเสียงพากย์ของเครื่องคุณให้อ่านออกเสียงทุกครั้งที่มีข้อความเด้งขึ้นมา

▶ พูดชัด อ่านซ้ำได้

คุณจะได้รับฟังการออกเสียง Pitch Accent ที่เป็นธรรมชาติที่สุด เพื่อเตรียมพร้อมหูก่อนเจอข้อสอบจริง

ตั้งค่าเสียงเกม

  1. เรียกใช้เสียงพากย์จากสมาร์ทโฟน
  2. เสียงอ่านคมชัดทุกลมหายใจ
  3. รับประกันการตอบสนองที่ไวสุดๆ

สโลว์สปีดเสียงให้พอดีกับความเข้าใจ (Rate)

สาวแกล (Gyaru)

「ゆっくり喋るね。聞いて。」
จะพูดช้าๆ นะ ตั้งใจฟังล่ะ

หากเสียงเจ้าของภาษาเร็วเกินไปจนจับใจความคำช่วยไม่ทัน คุณเข้าไปที่ตั้งค่าและปรับ Rate ให้น้อยกว่า 1 ได้อย่างอิสระ

▶ ฟังไม่ทัน ไม่ใช่ปัญหา

เป็นกุญแจสำคัญสำหรับคนที่เพิ่งเริ่มฝึกฟัง N4 เพราะการจับคำช่วยท้ายประโยคให้ทันคือส่วนสำคัญในข้อสอบระดับนี้

การลากแถบ Slider

  • ปรับความเร็วได้ลื่นไหล
  • ตัวเกมจะจดจำค่าที่คุณเลือกเสมอ
  • ช่วยให้จับผิดประโยคซ้อนประโยคได้

หน้าต่างสคริปต์ภาษาไทย คลายปมสงสัยฉับพลัน

สาวแกล (Gyaru)

「タイ語のスクリプトを確認してね。」
เช็คสคริปต์ภาษาไทยดูสิ

พาร์ทการฟังของโจทย์ทั่วไป มักเปิดให้ฟังอย่างเดียว แต่ในเกมนี้คุณสามารถอ่านกล่องข้อความด้านล่าง แบบไทยคู่ญี่ปุ่น

▶ แปลละเอียด

ทำให้เกิดการประมวลผลทั้งรูป เสียง และความหมาย ไปพร้อมๆ กันในการสบตาเดียว ไม่ต้องหาหนังสือเฉลย

การดูสคริปต์ข้อความ

  • ตัวอักษรคันจิพร้อมคำแปลไทยครบ
  • ดูได้ตลอดการเคลื่อนไหวของเกม
  • ร่นเวลาศึกษาประโยคยากๆ

อุดช่องโหว่ด้วยหมวด Communication

สาวแกล (Gyaru)

「基礎からやり直せるよ！」
เริ่มใหม่ตั้งแต่พื้นฐานได้เลยนะ!

หากคอร์สทดสอบ N4 ยังยากไป เกมขยับฐานของคุณด้วยคอร์ส Communication เพื่อฝึกการตอบรับแบบ Aizuchi

▶ เสริมฐานราก N4 แบบเนียนๆ

คุณสามารถฝึกฐานจากบทสั้น 1 คำถาม 1 คำตอบ ก่อนที่จะกระโดดเก็บคะแนนโหมด Listening เต็มตัว

คอร์สการฟังรองรับ

  1. แยกหมวดหมู่ชีวิตประจำวันชัดเจน
  2. ฝึกโต้ตอบคำถามแบบปิงปอง
  3. วิเคราะห์คำถาม A-B ให้เร็วขึ้น
